🔐 Custody & Wallet Infrastructure

Four security incidents this week: third-party module risk, multisig governance failure, private key theft, and DPRK supply-chain attack.

Squid Protocol Distances Itself From $3.2M Safe Wallet Module Exploit

What:A third-party module named SquidRouterModule drained ~$3.2M from Safe wallets. Squid says its core protocol was unaffected and it didn't deploy the module — yet it carried their name and hit production wallets.
Source:The Block  ·  CoinTelegraph
So what: Build an explicit allow-list registry and on-chain provenance check for any third-party modules touching client wallets. "We didn't deploy it" won't satisfy enterprise clients who lost funds.

StablR Multisig Compromise Drops EURR to $0.85, USDR to $0.40

What:Attacker exploited StablR's minting key via multisig compromise, minted $13.5M in unbacked tokens, dumped ~$10.4M on DEXes. Both stablecoins depegged severely.
Source:The Block  ·  The Defiant
So what: Minting-key governance is as critical as private-key security. Evaluate whether integrated stablecoins have independent reserve attestation that would flag unbacked minting before market damage occurs.

LayerZero Post-Mortem: DPRK Behind $292M Kelp Exploit via DVN Downgrade

What:Six-week breach traced to North Korea's TraderTraitor group. Root cause: Kelp downgraded its DVN from 2-of-2 to 1-of-1, eliminating the redundancy that would have caught the attack. LayerZero now defaults to 3-of-3.
Source:The Defiant
So what: Any validator or signer quorum parameter change should require a change-management process as rigorous as a key-rotation ceremony. Add this explicitly to your vendor onboarding checklist.

AI Is Accelerating the Quantum Timeline for Cryptographic Threats

What:Security researchers warn AI-assisted cryptanalysis is compressing the window before quantum computers can threaten elliptic-curve and RSA key schemes underpinning crypto wallets and custody systems.
Source:CoinDesk
So what: Document which key-generation and signing libraries your platform uses and verify they have a NIST PQC migration path. Institutional RFPs will start asking within 12–18 months.
